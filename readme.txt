== Description ==

This is a very easy plugin that uses jquery to provide an Accordion effect for the Getshopped categories widget.
It requires no settings.. Just upload the plugin and activate it to give you the accordion effect for the Categories widget. 
1- Exctract the files and put it in the plugins folder
2- Activate the plugin from the control panel.
3- Start adding the GetShopped Categories/Sub Categories. You will find that the Parent Category is now bold and when you click on it the subcategories will appear. 

== Installation ==



1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
